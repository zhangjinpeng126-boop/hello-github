# 教务管理系统（Student Management System）

Python 基础语法练习项目，基于字典实现的学生成绩管理系统。

## 已实现功能

- [x] 添加学生信息（姓名、数学、语文、英语成绩）
- [x] 修改学生信息
- [x] 删除学生信息
- [x] 查询学生信息
- [x] 列出所有学生
- [x] 统计班级成绩（最高分、最低分、平均分）
